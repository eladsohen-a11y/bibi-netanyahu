# מה ביבי היה אומר? 🎤

**אפליקציית סאטירה אינטראקטיבית** — תוכלו להוציא את הפוליטיקאי הגדול בכל הזמנים מהכלים?

7 שאלות. מד לחץ. תשובות חיות מ-AI. רגע הכרעה לסיום דרמטי. שתפו את התוצאה בוואטסאפ ותראו מי מנצח.

> ⚠ פרויקט פרודיה וסאטירה — התשובות נוצרות אוטומטית על ידי AI ואינן ציטוטים אמיתיים של אף אדם.

---

## איך זה עובד

- **המשתמש** שואל שאלה.
- **השרת** שולח אותה ל-Claude (Anthropic) עם system prompt שמכייל את האופי.
- **התשובה** חוזרת עם הבעת פנים (mood) ותמונה דרמטית מתאימה.
- **מד הלחץ** עוקב אחרי לחץ מצטבר לפי נושאי השאלה — עם תשואות פוחתות לחזרה על אותו נושא, בונוס לקומבו על שאלות מגוונות, והתנגדות מדורגת ככל שמתקרבים ל-100%.
- **שאלת הכרעה** מופעלת אוטומטית אם המשתמש נשאר על 90%–99% אחרי 7 שאלות.

---

## טכנולוגיה

- **Frontend:** HTML / CSS / JS טהורים. בלי framework.
- **Backend:** Node.js + Express.
- **AI:** Anthropic Claude (`@anthropic-ai/sdk`).
- **פריסה:** Render (פרי-טייר).

---

## הרצה מקומית

```bash
npm install
echo "ANTHROPIC_API_KEY=sk-ant-..." > .env
node server.js
```

פתחו `http://localhost:3000`.

המודל נקבע דרך משתנה סביבה `MODEL` (ברירת מחדל: `claude-haiku-4-5`). למשחק עם תשובות איכותיות יותר, הגדירו `MODEL=claude-sonnet-4-6`.

---

## אבטחה

- מפתח ה-API שמור ב-`.env` ולעולם לא מועלה ל-git.
- הגבלת קצב: 15 שאלות ל-10 דקות לכל IP.
- תקרה יומית גלובלית: 100 שאלות (ניתן להתאמה דרך `DAILY_CAP`) — מונע הוצאות מפתיעות.
